# latihansoalmudah1
Program Pengiriman Paket: Program ini menghitung biaya pengiriman berdasarkan berat, jarak, dan volume paket. Jika jaraknya ≤10 km maka tarifnya Rp 4250/kg, kalau lebih dari itu Rp 6000/kg. Kalau volume lebih dari 100 cm³ ada tambahan biaya Rp 50.000. Hasil akhirnya menampilkan total biaya kirim.
